# Vendored

This package is a **vendored snapshot** of the reference adapter, shipped *with the demo*
so `nilscript demo` works from a plain `pip install nilscript[demo]` (no separate install).

- Source: https://github.com/nilscript-org/pocketbase-nil-adapter
- Synced at: nilscript 0.3.0 (adapter commit `63dcebe`)
- It is a **demo file**, not a separately published package. Re-vendor on each release.
